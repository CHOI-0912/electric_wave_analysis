# 실행 메모

현재 코드는 폴더 위치를 내부 경로 기준으로 잡아둔 상태입니다.
어디서 실행해도 아래 경로를 사용합니다.

```text
입력: data/excel/*.xlsx
출력: output/
```

## 1. 파형 분석

```powershell
python script\plot_wave.py
```

하는 일:

- `data/excel/` 안의 xlsx 파일을 읽음
- 각 지점을 `rising`, `holding`, `falling`으로 분류
- `output/electric_wave_analysis [파일명].csv` 생성
- `output/electric_wave_analysis [파일명].png` 생성

## 2. holding 표준편차 계산

```powershell
python script\holding_std.py
```

하는 일:

- `output/electric_wave_analysis [파일명].csv`를 읽음
- `data/excel/`의 원본 파형 값을 같이 사용
- `output/holding_std.csv` 생성
- `output/nothing_ratio_sweep.csv` 생성
- `output/nothing_trim_sweep.csv` 생성

## 3. 보조 계산

```powershell
python script\hold_std.py
```

하는 일:

- holding 구간을 고정 margin 기준으로 잘라 표준편차 계산
- `output/hold std [파일명].csv` 생성
- `output/hold std [파일명].txt` 생성
- `output/hold std summary.csv` 생성

## 주의

`holding_std.py`와 `hold_std.py`는 먼저 파형 분석 CSV가 있어야 동작합니다.
처음부터 다시 만들 때는 `plot_wave.py`를 먼저 실행합니다.

기존 `output/` 파일과 이름이 같으면 새 결과가 덮어써집니다.
